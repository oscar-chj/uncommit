# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2024-12-29

### Added
- Initial release
- `uncommit analyze` - list uncommitted changes
- `uncommit suggest` - AI-powered commit grouping
- `uncommit commit` - commit suggested groups
- `uncommit undo` - undo last commit
- `uncommit status` - show cached suggestions
- `uncommit clear` - clear cache
